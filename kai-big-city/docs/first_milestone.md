# First Milestone

## Goal

Create a playable third-person prototype where Kai can move through a small city block, collect treats, and complete one tiny errand.

## Deliverables

- Unreal third-person starter project
- Graybox level starting from a home block inspired by Kai's first neighborhood, with one street, one alley, one rooftop route, and one pocket park
- Placeholder dog player
- Treat collectibles
- Basic interact prompt
- One NPC errand

## Feel Targets

- Fast, playful, responsive movement
- Small world with dense interaction
- Easy to understand goals
- Cute dog personality in movement and presentation

## Build Order

1. Project creation in Unreal
2. Third-person controller tuned for dog movement
3. Graybox neighborhood block
4. Treat collectible system
5. Interact prompt system
6. NPC errand
7. Basic audio and juice
